RAG // Embeddings

Agentic AI Assistant for Theme Park Operations

Multi-agent application designed to assist theme-park staff. Built the orchestration layer using Python, ADK, and FastMCP. Gemini autonomously decides which agents to call and tools to call. Designed the RAG pipeline around ChromaDB and Gemini's embedding model, using asymmetric query/document embeddings and a similarity-confidence threshold to prevent low-quality matches from reaching the model. Paired with a Vue 3 and TypeScript frontend.
